A chemistry-themed top-down educational game with a learning analytics dashboard that aimed to teach the user the knowledge of chemistry at the end of the educational gameplay. Read more... Within the game, the user can immerse through different worlds, where each world stands for a single chapter. For each stage, there'll be three ministages for the user to answer in order to complete the stage. The user would need to collect tips, ask for help or answer directly. As for the dashboard, the dashboard is aimed to improve the user gameplay experience by showing the statistics of the user's individual educational gameplay. Based on the statistics of all users, learning effectiveness of the assisting tools of the ministages is calculated based on the data collected from the users. Read less...
